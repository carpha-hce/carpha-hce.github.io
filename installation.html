Installing R and RStudio

HCE @ CARPHA

Last updated: 2022-11-24

1. Install R and RStudio

  • Download R @ The R Projects Home Page

    • Windows: https://cran.r-project.org/bin/windows/base/R-4.2.2-win.exe

    • Mac (ARM i.e.models manufactured since November 2020): https://cloud.r-project.org/bin/macosx/big-sur-arm64/base/R-4.2.2-arm64.pkg

    • Mac (Intel i.e.models manufactured prior to November 2020): https://cloud.r-project.org/bin/macosx/base/R-4.2.2.pkg

    • Linux https://cloud.r-project.org/bin/linux/

  • Download RStudio Desktop from RStudio

    • Windows https://download1.rstudio.org/desktop/windows/RStudio-2022.07.2-576.exe

    • Mac https://download1.rstudio.org/desktop/macos/RStudio-2022.07.2-576.dmg

    • Linux https://www.rstudio.com/products/rstudio/download/#download

Run the installers for R and RStudio, in that order.

3. Install additional packages and data

  • copy-paste the text below into the command window
  • hit enter
  • accept any defaults if you get prompted
  • this may take a little while. But nothing too exorbitant.
install.packages(c("tidyverse", "nycflights13"))

The picture below should show where you paste the command before hitting enter

Screenshot of RStudio a command in the console

4. Test it out

Copy-paste the following text into the command window and hit enter.

If the graphs show up then you have installed all the necessary packages. Dont worry if you dont understand the commands.

library(tidyverse)
library(nycflights13)
flights <- flights |> 
  mutate(cancelled = is.na(arr_delay) & is.na(dep_delay),
         Day = ISOdate(year, month, day))

flight_report <- flights |>
  group_by(Day, origin) |> 
  summarise(Cancelled = mean(cancelled)*100)

ggplot(flight_report, aes(x = Day, y = Cancelled)) +
  geom_line() +
  facet_wrap(~origin, ncol = 1) +
  labs(title = "Flight cancellations in NYC 2013", 
       y = "% of Flights Cancelled")
